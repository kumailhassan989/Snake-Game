# Snake Game

The **Snake Game** is a classic arcade-style game built entirely with **HTML, CSS, and JavaScript**. The objective is to control the snake to eat food, grow longer, and achieve the highest possible score. The game offers smooth and responsive controls, real-time score tracking, and intuitive game over and restart options for an engaging gameplay experience.
